Servidor

Um servidor é um computador ou sistema que armazena dados, arquivos e conteúdo online. Ele atende a solicitações de clientes, enviando informações quando requisitado.

Clientes

Os clientes são dispositivos, como computadores, smartphones ou tablets, que acessam conteúdo online. Eles solicitam dados a servidores por meio de navegadores ou aplicativos.

Página da Web

Uma página da web é um documento online acessado por meio de um navegador. Ela pode conter texto, imagens, vídeos e outros elementos interativos.

HTML

HTML (Hypertext Markup Language) é a linguagem de marcação usada para criar páginas da web. É composta por elementos estruturais que definem o conteúdo e sua apresentação.

Estrutura básica de uma página HTML

- <!DOCTYPE html>: Declara o tipo de documento HTML.
- <html>: Define o início e o fim do documento HTML.
- <head>: Contém metadados, como título da página e links para estilos e scripts.
- <title>: Define o título da página exibido na aba do navegador.
- <body>: Contém todo o conteúdo visível da página, como texto, imagens e links.
- <h1>, <p>, <img>, <a>: Exemplos de elementos HTML para cabeçalhos, parágrafos, imagens e links, respectivamente.
